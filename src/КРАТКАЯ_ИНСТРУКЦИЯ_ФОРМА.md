# ⚡ Краткая инструкция: Форма цен

## 🎯 Суть

**Редактируйте ЛЮБОЕ поле - остальные пересчитаются автоматически!**

```
┌─────────────────┬─────────────────┐
│ 💰 Цены         │ 📊 Комиссии     │
├─────────────────┼─────────────────┤
│ Розница: 6900   │ L0: 2000       │ ← Меняйте ЛЮБОЕ
│ Уровень 1: 4900 │ L1: 400        │
│ Уровень 2: 4500 │ L2: 900        │
│ Уровень 3: 3600 │ L3: 300        │
│ 3 линии: 3300   │ Компания: 3300 │
└─────────────────┴─────────────────┘
```

---

## 🔄 Примеры

### 1. Изменить розничную цену
```
Розница: 6900 → 7000
→ L0 изменится: 2000 → 2100
```

### 2. Изменить L1
```
L1: 400 → 600
→ Уровень 2 изменится: 4500 → 4300
```

### 3. Изменить доход компании
```
Компания: 3300 → 3500
→ L3 изменится: 300 → 100
```

---

## ✅ Проверка

Внизу должно быть:

```
Итого комиссий: ₽3,600 | Проверка: ✅ Правильно
```

Если **❌ Ошибка** - пересчитайте!

---

## 💡 Быстрый старт

### Способ 1: Сверху вниз
1. Розница: **6900**
2. Партнёрская: **4900** → L0 = 2000
3. Уровень 2: **4500** → L1 = 400
4. Уровень 3: **3600** → L2 = 900
5. 3 линии: **3300** → L3 = 300

### Способ 2: Снизу вверх (через комиссии)
1. Компания: **3300**
2. L3: **300** → Ур3 = 3600
3. L2: **900** → Ур2 = 4500
4. L1: **400** → Партнёрская = 4900
5. L0: **2000** → Розница = 6900

---

## 🎨 Цвета

- 🟢 Зелёный = L0 (доход от гостя)
- 🔵 Синий = L1 (спонсор 1 линии)
- 🟠 Оранжевый = L2 (спонсор 2 линии)
- 🟣 Фиолетовый = L3 (спонсор 3 линии)
- ⚫ Серый = Компания (остаток)
