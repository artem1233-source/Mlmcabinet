# 📸 Пример из скриншота

## Исходные данные со скриншота

```
┌────────────────────────┬────────────────────────┐
│ 💰 Ценообразование     │ 📊 Комиссии            │
├────────────────────────┼────────────────────────┤
│ Розничная: 6900       │ L0: ₽2,000            │
│ Цена для гостей        │ = Розница - Партнёрская│
├────────────────────────┼────────────────────────┤
│ Уровень 1: 4900       │ L1: ₽400              │
│ Партнёрская цена       │ = Ур1 - Ур2            │
├────────────────────────┼────────────────────────┤
│ Уровень 2: 4500       │ L2: ₽900              │
│ Для расчёта L1         │ = Ур2 - Ур3            │
├────────────────────────┼────────────────────────┤
│ Уровень 3: 3600       │ L3: ₽300              │
│ Для расчёта L2         │ = Ур3 - Ур4            │
├────────────────────────┼────────────────────────┤
│ 3 линии: 3300         │ Компания: ₽3,300      │
│ Базовая цена компании  │ Остаток после выплат   │
└────────────────────────┴────────────────────────┘
         Итого комиссий: ₽3,600 | ✅ Правильно
```

---

## 🔍 Проверка расчётов

### Комиссии:
```
L0 = 6900 - 4900 = 2000 ✅
L1 = 4900 - 4500 = 400 ✅
L2 = 4500 - 3600 = 900 ✅
L3 = 3600 - 3300 = 300 ✅
```

### Сумма:
```
L0 + L1 + L2 + L3 + Компания = Розница?
2000 + 400 + 900 + 300 + 3300 = 6900 ✅
```

### Итого комиссий:
```
L0 + L1 + L2 + L3 = ?
2000 + 400 + 900 + 300 = 3600 ✅
```

---

## 🔄 Примеры изменений

### Пример 1: Изменить L0 с 2000 на 2500

**Меняем L0 (справа):**
```
L0: 2000 → [2500]
```

**Автоматически пересчитывается Розница (слева):**
```
Было:
Розница: 6900
L0: 2000

Стало:
Розница: 7400 ← автоматически (2500 + 4900)
L0: 2500 ✅
```

**Новая проверка:**
```
2500 + 400 + 900 + 300 + 3300 = 7400 ✅
```

---

### Пример 2: Изменить L1 с 400 на 600

**Меняем L1 (справа):**
```
L1: 400 → [600]
```

**Автоматически пересчитывается Уровень 2 (слева):**
```
Было:
Партнёрская: 4900
Уровень 2: 4500
L1: 400

Стало:
Партнёрская: 4900
Уровень 2: 4300 ← автоматически (4900 - 600)
L1: 600 ✅
```

**Влияние на L2:**
```
Было:
L2 = 4500 - 3600 = 900

Стало:
L2 = 4300 - 3600 = 700 ← уменьшилось!
```

**Новая проверка:**
```
2000 + 600 + 700 + 300 + 3300 = 6900 ✅
```

---

### Пример 3: Изменить L2 с 900 на 1000

**Меняем L2 (справа):**
```
L2: 900 → [1000]
```

**Автоматически пересчитывается Уровень 3 (слева):**
```
Было:
Уровень 2: 4500
Уровень 3: 3600
L2: 900

Стало:
Уровень 2: 4500
Уровень 3: 3500 ← автоматически (4500 - 1000)
L2: 1000 ✅
```

**Влияние на L3:**
```
Было:
L3 = 3600 - 3300 = 300

Стало:
L3 = 3500 - 3300 = 200 ← уменьшилось!
```

**Новая проверка:**
```
2000 + 400 + 1000 + 200 + 3300 = 6900 ✅
```

---

### Пример 4: Изменить L3 с 300 на 500

**Меняем L3 (справа):**
```
L3: 300 → [500]
```

**Автоматически пересчитывается Цена 3 линии (слева):**
```
Было:
Уровень 3: 3600
3 линии: 3300
L3: 300
Компания: 3300

Стало:
Уровень 3: 3600
3 линии: 3100 ← автоматически (3600 - 500)
L3: 500 ✅
Компания: 3100 ← тоже изменилась!
```

**Новая проверка:**
```
2000 + 400 + 900 + 500 + 3100 = 6900 ✅
```

---

### Пример 5: Изменить Доход компании с 3300 на 3500

**Меняем Компанию (справа):**
```
Компания: 3300 → [3500]
```

**Автоматически пересчитывается всё:**
```
Было:
Уровень 3: 3600
3 линии: 3300
L3: 300
Компания: 3300

Стало:
Уровень 3: 3600
3 линии: 3500 ← автоматически
L3: 100 ← пересчитался (3600 - 3500)
Компания: 3500 ✅
```

**Новая проверка:**
```
2000 + 400 + 900 + 100 + 3500 = 6900 ✅
```

## 💡 Что можно делать

### ✅ Редактировать ЦЕНЫ (слева):
- Розничная цена → пересчитается L0
- Уровень 1 → пересчитаются L0 и L1
- Уровень 2 → пересчитаются L1 и L2
- Уровень 3 → пересчитаются L2 и L3
- 3 линии → пересчитается L3

### ✅ Редактировать КОМИССИИ (справа):
- L0 → пересчитается Розничная цена
- L1 → пересчитается Уровень 2
- L2 → пересчитается Уровень 3
- L3 → пересчитается 3 линии
- Компания → пересчитаются 3 линии и L3

## 🎯 Практическое применение

### Задача: Установить комиссии L0=2500, L1=500, L2=800, L3=400

**Решение (через комиссии справа):**
```
Шаг 1: Установите базу
Компания: [3000]

Шаг 2: Установите L3

Шаг 3: Установите L2

Шаг 4: Установите L1
L1: [500] → Партнёрская = 4700

Шаг 5: Установите L0
L0: [2500] → Розница = 7200

РЕЗУЛЬТАТ:
Розница: 7200
Партнёрская: 4700
Уровень 2: 4200
Уровень 3: 3400
3 линии: 3000

L0: 2500 ✅
L1: 500 ✅
L2: 800 ✅
L3: 400 ✅
Компания: 3000 ✅

Проверка: 2500+500+800+400+3000 = 7200 ✅
```
