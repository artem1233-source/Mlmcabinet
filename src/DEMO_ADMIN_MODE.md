# 👑 Демо-режим Администратора

## ✅ Что изменено

### Теперь в превью Figma Make автоматически показывается **Админ-кабинет**!

---

## 🎯 Изменения

### 1. **Демо-данные** (`/utils/demoData.ts`)

Главный пользователь теперь **администратор**:

```typescript
const mainUser = {
  id: 'DEMO_USER',
  telegramId: 999999999,
  email: 'admin@admin.com',      // ✅ Добавлен email админа
  имя: 'Администратор',           // ✅ Изменено имя
  фамилия: 'Системы',             // ✅ Изменена фамилия
  username: 'admin',
  уровень: 3,                     // ✅ Максимальный уровень
  рефКод: 'ADMIN2024',
  isAdmin: true,                  // ✅ АДМИН!
  глубина: 0
};
```

### 2. **Кнопка "Демо-вход"** (`/components/EmailAuthRu.tsx`)

Теперь активирует **полноценный демо-режим**:

```typescript
const handleDemoLogin = async () => {
  // Генерируем демо-данные с админом
  const demoData = generateAllDemoData();
  
  // Сохраняем в localStorage
  saveDemoDataToStorage(demoData);
  
  // Устанавливаем токен демо-режима
  setAuthToken('DEMO_TOKEN');
  
  // Логинимся как админ
  onAuth(demoData.currentUser);
};
```

**Было:** вызывал сервер и создавал обычного пользователя  
**Стало:** создаёт локальные демо-данные с админом

## 🚀 Как это работает

### 1. **Открываете превью Figma Make**
   - Видите экран входа

### 2. **Нажимаете кнопку "👑 Войти как Администратор (демо)"**
   - Генерируются демо-данные
   - 50+ пользователей
   - 100+ заказов
   - 6 месяцев истории

### 3. **Автоматически входите как админ**
   - Email: admin@admin.com
   - isAdmin: true
   - Полные права

### 4. **Видите админ-интерфейс**
   - 👑 Золотой badge "Режим администратора"
   - 🌍 Глобальная статистика в Дашборде
   - ➕ Управление товарами в Каталоге
   - 💰 Управление выплатами в Балансе
